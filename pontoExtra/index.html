HTML



HTML funciona por meio de tags como p (parágrafos), h1 até h6 (títulos), a (links), img (imagens), ul, ol e li (listas). A estrutura básica de uma página começa com , seguida por html, head (informações da página como título, meta e links) e body (conteúdo visível). Formulários são criados com form, input, textarea e button. Para organizar dados em tabelas, usamos table (tabela), tr (linha), td (célula comum) e th.

CSS

CSS é usado pra dar estilo às páginas, deixando o visual separado do HTML. Com ele dá pra mudar cores, fontes, margens, espaço entre os elementos, posição e até fazer animações. Principais tags:
style: coloca CSS direto no HTML. body: define o estilo geral da página. div: usada pra organizar e agrupar partes da página. .classe: aplica o mesmo estilo em vários elementos. #id: aplica estilo em um elemento específico.

GIT



O Git é um sistema de controle de versão distribuído usado para gerenciar alterações no código de um projeto. Ele facilita o trabalho em equipe, permitindo que vários desenvolvedores colaborem sem sobrescrever o código uns dos outros Principais comandos:
git init (inicializa um repositório Git) git clone (copia um repositório existente) git add (adiciona arquivos à área de preparação) git status (mostra o estado atual do repositório) git push (envia as mudanças para o repositório remoto) git pull (atualiza o repositório local com mudanças do remoto)

GITHUB

GitHub é um serviço online voltado para o armazenamento e gerenciamento de códigos, baseado no sistema Git. Ele permite que desenvolvedores trabalhem juntos em projetos, façam alterações de forma segura e compartilhem seus códigos com outras pessoas. Principais funcionalidades:
Repositórios: Uma forma de armazenar códigos e compartilhar para terceiros. Fork: cópia de um repositório para editar separadamente. Pull Request: pedido para enviar mudanças ao repositório principal. Issues: registro de tarefas, erros ou sugestões.

JAVASCRIPT

JavaScript é uma linguagem usada no front-end e também no back-end com Node.js. No front, ele deixa a página interativa, respondendo a cliques, digitação e outros eventos. Pode ser usado junto com PHP e HTML para criar sites dinâmicos, trocando dados sem precisar recarregar a página. Principais recursos:
Manipulação do DOM: mudar ou acessar elementos do HTML direto na página. Eventos: reage a ações do usuário, tipo clique ou tecla. AJAX: pega ou envia dados sem recarregar a página. Funções: pedaços de código que dá pra usar várias vezes
